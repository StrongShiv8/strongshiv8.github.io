So lets use the hashcat rules here and lets create the password file →

```bash
┌──(kali㉿kali)-[~/Downloads/HTB/Delivery]
└─$ hashcat -r /usr/share/hashcat/rules/best64.rule --stdout file > password.txt
```
{: .nolineno}
